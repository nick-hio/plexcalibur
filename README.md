# plexcalibur

Utility for your Plex server to automatically notify you (via webhooks) when new media is available.

## Tools and Libraries

| Tool/Library                        | Usage                |
|-------------------------------------|----------------------|
| [**Bun**](https://bun.sh/)          | Runtime and bundler. |
| [**Fastify**](https://fastify.dev/) | HTTP server.         |
